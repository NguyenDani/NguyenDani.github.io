Client-Server

This project involves the development of a multi-threaded echo server and a corresponding client application in C++. The server listens for incoming client connections on a specified port, and for each connected client, it spawns a new thread to handle communication. The server echoes back any data it receives from the clients, demonstrating the use of socket programming, multi-threading, and inter-process communication. Key skills showcased in this project include network programming with TCP/IP, utilizing system calls, and multi-threading to manage concurrent client connections efficiently. Error handling and data transmission management are also integral parts of the implementation, hightlighting a strong understanding of both low-level system programming and high-level application design. Github

Market-Analysis

The Market Data Analysis Tool is a sophisticated application designed to analyze historical market data and identify patterns, trends, and correlations within financial data. By integrating both C++ and Python, this tool leverages the computational efficiency of C++ for real-time data processing and the flexibility of Python for data visualization. The application fetches real-time market data from Yahoo Finance, performs various statistical analyses such as moving averages, Bollinger Bands, and MACD, and presents the results through interactive plots. Utilizing multithreading for concurrent data processing, this tool ensures efficient handling of large datasets, making it an invaluable resource for traders, financial analysts, and researchers looking to gain actionable insights from market trends.
